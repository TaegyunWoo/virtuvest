## 1. 기본 주가 데이터

### 1. 국내 주식 (KOSPI, KOSDAQ)
| 데이터 항목        | 설명            | 용도            |
| ------------- | ------------- | ------------- |
| 종가 (Close)    | 매일 거래 종료 시 가격 | 수익률 계산, 차트 분석 |
| 시가 (Open)     | 거래 시작 시 가격    | 일중 변동 분석      |
| 고가 (High)     | 당일 최고가        | 기술적 분석        |
| 저가 (Low)      | 당일 최저가        | 기술적 분석        |
| 거래량 (Volume)  | 거래된 주 수        | 시장 심리 분석      |
| 거래대금 (Amount) | 총 거래 금액       | 유동성 판단        |
| 시가총액          | 기업의 총 가치      | 대형주/중소형주 분류   |
| 상장주식수         | 발행된 총 주 수      | 가치평가 기초       |

### 2. 실시간 시세 데이터
- **수집 주기**: 거래 시간 내 실시간 (장 종료 후 1회 별도 수집)
- **대상**: 서비스에서 취급하는 모든 국내 주식
- **용도**: 가상 투자 시 실제 시장가 적용, 변동성 체험 제공

## 11. 우선순위 정의

### 1단계 (MVP 필수)
- 국내 주식 실시간 시세 (KOSPI, KOSDAQ)
- 기본 기업 정보 (종목명, 코드, 업종)
- 매수/매도 거래 기능
- 포트폴리오 현황 조회

### 2단계 (확장)
- 기술적 분석 지표
- 재무정보
- 해외 시장 지수

### 3단계 (고급 기능)
- AI 기반 분석
- 자동화된 투자 전략 추천
- 고급 기술적 패턴 인식
